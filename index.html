NFL Game Predictor

For my CMSC320 Data Science course, my final project involved doing analysis on data of past NFL games. During my exploratory analysis, I examined different factors such as home field advantage, rushing, passing yards, etc. to see what effect they had on the outcome of games. At the end, I fed the data into the scikit random forest library to predict games. This tutorial project can be found at https://braydenngo.github.io/cmsc320finalngo/ngo_final_project%20(3).html

Byte Buddies

This was a project my group and I were building for Bitcamp 2023. While we didn't end up finishing we still had a great time working together at our first hackathon! Our idea was to create a web application that would make apartment and housing easier for new interns. We always see interns on sites like reddit and discord making posts about DMing for other interns that are in the same area for housing for their upcoming internship so we wanted to make an application that would make this process easier. For our tech stack, we ended up using HTML, CSS, and Javascript for the front end and the Django framework for our backend. In order to host the website, we attempted to use Microsoft Azure. You can find the Github repo at the following link: https://github.com/BraydenNgo/ByteBuddies
